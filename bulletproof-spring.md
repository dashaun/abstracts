# Bulletproof Your Spring Apps

Learn how to elevate your enterprise development with Spring and Spring Boot.
Learn how Spring Boot helps improve your software supply chain and new actuator features.
Learn how to keep your Spring and Spring Boot applications patched and performant, with OpenRewrite recipes.
Say 'goodbye' to Docker headaches with and say 'hello' to Cloud Native Buildpacks.  Because friends don't let friends user Dockerfile.
We will wrap with an overview of AOT processing, class data sharing, and GraalVM.
